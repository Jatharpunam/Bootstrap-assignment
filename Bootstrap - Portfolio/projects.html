WhatsApp Chatbot

Developed a WhatsApp chatbot for the college admission process using Python, ReactJS, and MongoDB. The chatbot assists students with queries on courses, deadlines, admission criteria, and application procedures. It offers a user-friendly platform accessible across devices. Smart Robo Car - An Advanced Object Avoiding Robot

Developed an advanced obstacle-avoiding robot with Bluetooth and voice control on the Arduino platform, designed for individuals with limited mobility. The robot uses ultrasonic sensors for safe navigation around obstacles. Voice control enables hands-free operation, while Bluetooth allows remote control via smartphones. The project enhances accessibility, ease of use, and freedom for users.

Microservices CI/CD Pipeline on Cloud Computing (QR Code Generator)

Developed a cloud-based QR Code Generator application using a microservices architecture. Implemented a robust CI/CD pipeline to automate build, test, and deployment processes. Utilized Docker for containerization, Kubernetes for orchestration, and GitHub Actions/Jenkins for continuous integration and deployment. Hosted on cloud platforms (e.g., AWS/GCP/Azure), ensuring scalability, high availability, and efficient resource management.
